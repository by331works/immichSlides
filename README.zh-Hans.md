## 照片已经被保存好了，但你真的还会看见它们吗？

Immich 很适合管理照片库。手机、相机、旅行、家庭、孩子、宠物、作品集，终于有了一个长期保存和整理的地方。

但保存好之后，照片很容易继续沉睡。

你知道它们在服务器里，也知道它们被归档、识别、备份了。可平时家里的电视、桌上的 iPad、床头的 iPhone，仍然很少主动把这些回忆带到眼前。

immichSlides 想解决的是后半段：**不是再做一个完整相册客户端，而是给你的 Immich 照片库一个真正会被看见的屏幕。**

## 为什么值得装在你的 Apple 设备上

### 把 Apple TV、iPad、iPhone 都变成 Immich 照片展示屏

Apple TV 适合客厅和家庭聚会，iPad 适合书桌、边柜和床头，iPhone 适合临时连接、预览和随身播放。

打开 immichSlides，连接自己的 Immich server，选择相册、人物或完整图库，就可以让照片在这些 Apple 设备上自然轮播。

### 不用再为轮播单独整理一套图片

你不需要为了做一个轮播屏，重新导出照片、手动挑图、维护另一个相册，或把照片迁移到新的云相册。

immichSlides 直接使用 Immich 里已有的相册和人物。你在 App 里选择播放范围，App 负责轮播。

### 比普通电子相框更适合已有照片库

如果家里已经有 Apple TV、iPad 或旧 iPhone，它们的屏幕和生态往往比普通电子相框更适合展示照片。

immichSlides 让这些设备直接连接你的 Immich 照片库：电视可以给家人一起看，iPad 可以成为常驻相框，iPhone 可以随时打开播放。

## 还有这些细节

### 共享设备，也保留控制权

Apple TV、iPad、iPhone 都可能是家人共用的设备。

immichSlides 支持用 6 位 PIN 保护 Immich 连接和播放范围，避免别人随手改设置、切换图库，或看到你不想展示的照片范围。

### 摄影信息：照片不只是画面

如果你希望照片播放时保留更多摄影细节，immichSlides 可以显示相机、镜头、焦距、光圈、快门、ISO、拍摄日期和地点等信息。

对摄影爱好者来说，这些信息本身也是回忆的一部分。

### 免费，无广告

当前版本免费，没有广告。

immichSlides 的目标很简单：给已经拥有 Immich 的人，一个更轻松、更适合 Apple 设备的照片展示层。

## 适合这些用户

- 已经在使用 Immich 的自托管照片用户；
- 想把 Apple TV、iPad 或 iPhone 变成照片展示屏的人；
- 想让旧 iPad 或 iPhone 继续发挥价值的人；
- 拥有多年照片库、但很少重新翻看的人；
- 想按相册、人物或完整图库轮播的人；
- 希望减少共享屏幕尴尬乱入的人；
- 不想把照片再交给另一个云相册服务的人。

## 它不是什么

immichSlides 不是 Immich 官方 App，也不隶属于 Immich 项目。

它不是完整 Immich 客户端，不提供照片上传、删除、编辑、备份、托管或社交分享。

它只专注一件事：**把你自己的 Immich 照片库，变成 Apple TV、iPad 和 iPhone 上更容易被看见的轮播体验。**

使用它需要你拥有自己的 Immich server URL 和 API key。
