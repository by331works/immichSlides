相册最后一页

常见问题，留在最后一页。

01支持哪些 Apple 设备？

支持 iPhone、iPad 和 Apple TV。它的重点不是管理照片，而是把你自己 Immich 里的照片放到家里的屏幕上轮播。

适合客厅电视、床头旧 iPhone、桌面 iPad。
02需要自己的 Immich server 吗？

需要。immichSlides 不提供云相册、照片托管或公共 demo 服务。你需要已经有，或愿意搭建自己的 Immich server。

没有 Immich，就先不要把它当普通相册 App 下载。
03怎么连到 Immich？

在 App 里输入自己的 Immich server URL 和 API Key，通过连接测试后读取照片。网站不会向你索取这些信息。

连接信息只属于你的设备和你的 Immich 服务。
04可以当数码相框用吗？

可以。你可以把 Apple TV、闲置 iPad 或 iPhone 用作照片轮播屏幕，并在 App 内选择随机、相册或人物范围。

它不是照片管理器，核心就是把 Immich 播到屏幕上。
05会上传或托管照片吗？

不会。照片来自你的 Immich 服务；immichSlides 不上传、托管、备份、删除、编辑或分享照片。

这条边界是这页最重要的注脚。
06是 Immich 官方 App 吗？

不是。immichSlides 是独立开发的 Immich 连接展示和轮播 App，不是官方 Immich App，也不是完整 Immich 客户端。

避免把它理解成官方客户端或云服务。
07Immich 相框和 immichSlides 是同一个 App 吗？

是。同一个 App 的中文名是「Immich 相框」，英文名是「immichSlides」。英文用户也可能把它理解成 Immich slides / Immich slideshow app；它连接你自己的 Immich server，用于照片轮播和家庭屏幕展示。

中文应用市场展示中文名，英文页面继续使用英文名。
